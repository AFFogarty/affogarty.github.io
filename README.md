# Andrew Fogarty - Personal Website

This repository hosts the personal website of Andrew Fogarty, a Software Engineer. The site is built using Jekyll and hosted on GitHub Pages.

## About

This is a simple, clean personal website that provides professional contact information and links to social profiles.

**Live Site:** [www.andrewfogarty.net](https://www.andrewfogarty.net)

## Technology Stack

- **Jekyll** - Static site generator
- **GitHub Pages** - Hosting platform
- **Bootstrap** - CSS framework for styling
- **Kramdown** - Markdown processor

## Local Development

To run this site locally:

1. **Prerequisites**
   - Ruby (2.7 or higher recommended)
   - Bundler (`gem install bundler`)

2. **Installation**
   ```bash
   git clone https://github.com/AFFogarty/affogarty.github.io.git
   cd affogarty.github.io
   bundle install
   ```

3. **Run the development server**
   ```bash
   bundle exec jekyll serve
   ```

4. **View the site**
   - Open your browser to `http://localhost:4000`
